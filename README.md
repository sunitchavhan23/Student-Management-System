# Student Management System

A simple **Student Management System** built using **Java Swing** and **JDBC**.  
This project helps users manage student records with features like adding, viewing, updating, deleting, and searching students. It also includes user login and registration functionality.

---

## Features

- User Registration and Login
- Add Student Details
- View All Students
- Update Student Information
- Delete Student Records
- Search Student by Name
- Dashboard with Menu Navigation

---

## Technologies Used

- Java
- Java Swing (GUI)
- JDBC
- MySQL Database

---

## How to Run

1. Install Java and MySQL.
2. Add MySQL JDBC Driver to the project.
3. Create the database and tables.
4. Configure `DBConnection.java`.
5. Run `RegisterForm.java` or `LoginForm.java`.

---

## Important Note

Enter the enrollment date in `YYYY-MM-DD` format.
